üîß TROUBLESHOOTING
================================================================================

Problem: Servers won't start
Solution: Run setup-local.bat first, then start-local.bat

Problem: Port already in use
Solution: Change PORT in backend/.env to 3001

Problem: Can't access localhost
Solution: Wait 20 seconds, check terminal windows are running

Problem: API not responding
Solution: Make sure backend is running on port 3000
